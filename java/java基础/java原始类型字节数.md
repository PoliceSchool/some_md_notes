java中的原始类型有8种，分别是

* boolean：一个字节，8比特
* byte: 一个字节，8bit
* char：两个字节，16比特
* short：两个字节，16比特
* int：四个字节，32比特
* float：四个字节，32比特
* long：八个字节，64比特
* double：八个字节，64比特

值得注意的是：虽然boolean, byte, char, short有自己的字节长度，但是由于JVM的指令条数限制，所以使用int类型的指令来调用这四个类型，所以在虚拟机内部，这四个类型其实都占用4个字节的长度，因为在虚拟机栈的局部变量表的每个slot就是4个字节长的。
